How It Works

GitHub Pages deploys this docs/ folder. A workflow copies key Markdown files from repo root into docs/knowledge/ on each push to main.

Pages Setup

  1. Open repository settings.
  2. Go to Settings → Pages.
  3. Set source to GitHub Actions.
